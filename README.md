JK Calendar
-----------

**Description**

We are recreating google Calendar.




### How to run locally

Create a virtual environment 

```
> virtualenv env
> source env/bin/activate
```

Install the dependencies

```
> pip install -r requirements.txt
```

Create Postgres Database
```
> python seed.py
```
